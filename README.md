# Week 1 JavaScript Assignments

This folder contains simple JavaScript programs for practicing basic programming concepts like arrays, functions, loops, conditions, and objects.

---

# Day 1 Tasks

### task1.js
Finds the total sum of numbers in an array.  
Also finds the smallest number in another array.

### task2&3.js
Checks which number is bigger between two numbers.  
Also finds the biggest number among three numbers.

### task4_.js
Works with employee data using arrays and objects.  
It shows how to add, remove, and update employee details.

### task5_.js
Creates a function that takes 3 numbers and returns the biggest number.

### task6_.js
Creates a function that receives an array and calculates the total sum of all numbers.

### task7.js
Searches for a number inside an array and returns its index position.

### task8.js
Finds the smallest number in an array using a loop.

# Concepts Used

- Variables (`let`)
- Arrays
- Objects

- If-Else Conditions
- For Loops
- Array Methods (`splice`)
